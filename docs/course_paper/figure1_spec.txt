CAPTION TEXT
================================================================================

"Figure 1: Overview of the TCD methodology and experimental pipeline.
(A) A query is scored against two counterfactual documents differing only
in an identity marker (Emily → Lakisha) using ColBERT's MaxSim.
(B) Per-token contributions are compared to compute Token Contribution
Disparity (TCD); function words (blue) show systematically higher TCD
than content words (orange).
(C) The Func-TCD signal serves as the response variable for three
downstream analyses: rarity amplification (§4.2), confound decomposition
(§4.3), and ecological validation (§4.4)."
