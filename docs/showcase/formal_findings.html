正式研究发现

ColBERT Bias Audit · 面向正式汇报的结果版总结 · 数据状态截至 2026 年 3 月 26 日

一句话总括：我们目前最稳的发现，不只是“模型会受身份信息影响”，而是“这种影响可以被定位到词级别，而且主要通过功能词传播”。

在此基础上，P1 说明这个现象在更大范围职业中仍然稳定存在；P2 进一步说明，经典的名字替换实验测到的并不是单一的“种族效应”，而是职业语境、名字的语言可处理性，以及身份类别残余效应共同作用的结果。

P1 正式发现：功能词偏见传导不是只在 10 个职业里偶然出现的现象，而是在更大职业集合中相当稳定的模式。

我们现在可以正式说什么

  • 功能词主导模式在 33 个职业里有 30 个成立，说明它具有跨职业的稳健性。
  • 把 name swap 和 pronoun swap 分开后，二者都保留了“功能词大于内容词”的模式；而且 pronoun swap 在 token level 上更强。
  • 职业类别的描述性差异仍然存在，但在当前 profession-level 的非参数检验里，类别间差异还没有被干净地拉开到显著。
  • 职业现实中的女性占比，与 pronoun-based 偏差更相关；与 name-based 偏差则没有形成稳定相关。

P2 正式发现：在扩展到 33 个职业、3 个模板、43065 次测试之后，嵌套模型、模板鲁棒性、leave-one-race-pair-out 敏感性分析，以及新增的正交子集面板共同表明：名字效应不是单一 race effect。最稳定的信号来自 cross-gender；在严格匹配子集中，tokenization 也会清晰保留；而 cross-race 更像是弱的、主要出现在 token level 的残余信号。
结果层级 统计结果 当前解释
总分偏差 SS cross_race p = 0.400，不显著；cross_gender p < 1e-14，显著 控制项加入后，跨种族名字对不再稳定预测总分偏差；但跨性别名字对会显著增大总分差异。
功能词扰动 Func-TCD cross_race p = 0.124，不显著；cross_gender p < 1e-13，显著 在功能词层面，稳定信号主要来自 gender crossing，而不是跨种族本身。
频率差异 freq_diff 对 SS 和 Func-TCD 都显著 在完整主模型中，名字之间的频率差距越大，偏差越容易放大；但在严格 matched subset 里，这个效应会明显减弱。

关于“名字到底是什么在起作用”

当前最稳妥的答案：名字效应不是单一因素。就目前证据看，它至少同时包含三层东西：
1. 职业语境带来的基线差异
2. 名字之间的频率差异所带来的语言不均衡
3. 性别相关的身份对比信号
  • 频率 / 罕见性：当前最稳定的 rarity 相关信号不是“名字整体越稀有越糟”。完整主模型里 freq_diff 稳定显著，但在同 race、同 gender、同 token count 的严格 matched subset 里，frequency-gap 效应会减弱到不稳定。这说明它更像是一个在整体样本中稳定存在、但不完全独立的解释项。
  • tokenization：升级版 P2 重新用真实 tokenizer 复核了名字切分，发现很多原先手工估计为多 token 的名字其实是 1 token。主模型里 token_diff 和 mean_tokens 不显著，但在严格匹配 race、gender 和 frequency 的子集中，token_diff 对 SS 和 Func-TCD 都会重新变成显著正向预测因子。
  • 种族：目前没有足够证据支持“cross-race 本身”是完整模型中的稳定主效应。它在简化模型和个别模板里会出现，leave-one-race-pair-out 也不能把它恢复成稳定主效应；但在严格 matched 的 race-isolated subset 里，cross-race 对 Func-TCD 仍保留了一个弱但可检出的 token-level 残余信号。
  • 性别：在升级版 P2 里，cross_gender 对 SS 和 Func-TCD 都是稳定显著的，而且在 same-race 子样本、以及更严格 matched 的 gender-isolated subset 里仍然成立。这说明名字实验里确实存在强的 gender contrast 信号。
  • SES：SES 已经进入主模型，但目前更像辅助控制项，而不是能单独支撑核心主张的主因。
最适合对外表述的版本：P2 不是在否定名字效应，而是在重新分配解释权。升级版结果更支持这样的表述：名字实验测到的是一个复合效应，其中 gender contrast 是最稳定的部分；tokenization 在严格匹配子集中会被干净看见；frequency gap 在完整样本中稳定但在正交子集里会减弱；独立的 cross-race 主效应并不稳固，只留下弱的 token-level 残余。

P2b 的第一轮 tokenizer-verified matrix 验证已经启动。首轮 targeted validation 与上面的主结论方向一致：gender 仍然最稳，race 仍主要表现为弱的 token-level 残余；但由于当前真正的 tokenization-treatment pair 只有 4 组，P2b 还需要通过扩充 verified 名字池来完成最后的 tokenization / absolute-rarity 检验。

新增的外部辅助验证也支持这一点：在一个不带 race 标签、只使用 country + gender + popularity rank 的外部名字池里，tokenization effect 再次显著复现（SS p = 0.0081, Func-TCD p = 3.0e-05）；而 popularity-rank-gap 本身没有形成稳定独立效应。这说明 tokenization 现在已经不只依赖当前 audit-name 集合，而是有了额外的数据源支撑。

可直接使用的结论句

关于 P1：

“我们把最初只在少数职业上看到的现象扩展到了 33 个职业，发现功能词偏见传导在 name swap 和 pronoun swap 中都稳定存在，而且 pronoun swap 在 token level 上更强，因此这不是偶然现象，而是 ColBERT 中一个相当稳定的结构性模式。”

关于 P2：

“我们进一步拆分了名字替换实验到底在测量什么。结果显示，在完整模型里，cross-race 不再是稳定显著的解释变量；最稳定的信号来自 cross-gender，而名字之间的频率差异也会在整体样本中放大偏差。与此同时，在严格匹配的子集中，tokenization 效应会清晰出现，而 cross-race 只保留了弱的 token-level 残余。这说明名字实验测到的不是单一 race effect，而是多种因素共同作用。”

关于名字因素的总判断：

“目前最稳的结论不是‘就是种族’或者‘就是罕见性’，而是：名字效应是职业语境、名字可处理性和身份类别残余效应共同构成的复合现象。”

还不能说得太满的地方

  • 还不能把 P2 讲成“已经证明存在稳定独立的 race main effect”，因为这个结论在完整规格、模板鲁棒性和敏感性分析里都不稳；当前更像是弱的 token-level residual。
  • 还不能把 P2 讲成“绝对罕见性不重要”，因为当前最稳定的是整体样本中的 frequency-gap 效应，而不是名字整体平均频率。
  • 还不能把 tokenization 讲成“已经被完全识别”，因为它目前主要在严格匹配子集中显现；要想彻底封口，还需要重新构造一版经真实 tokenizer 验证的名字矩阵。

换句话说：P2 已经把“经典 audit name swap 测量的是什么”推进成了一个相当成熟的机制性叙事。当前唯一真正还没封口的地方，是要用一套经过真实 tokenizer 验证的新名字矩阵，把 tokenization 和绝对 rarity 再做最后一次干净拆分。
