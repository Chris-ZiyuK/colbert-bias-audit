### 1. PDF 里有 Type 3 fonts，可能被提交系统或 camera-ready 检查卡住

我检查到 PDF 里有多个 **Type 3 fonts**，主要来自图里的 DejaVuSans / HelveticaNeue 等。很多 NLP/ACL 系提交系统或 camera-ready 检查会对 Type 3 fonts 很敏感。建议你在所有 matplotlib 出图脚本开头加：

```python
import matplotlib as mpl
mpl.rcParams["pdf.fonttype"] = 42
mpl.rcParams["ps.fonttype"] = 42
mpl.rcParams["font.family"] = "DejaVu Sans"
```

然后重新导出 figures、重新编译 PDF，再用：

```bash
pdffonts main.pdf
```

确认没有 Type 3。这个比微调一句话更重要。

### 3. TCD 的 “signed vs absolute” 说法不完全一致

现在 Methodology 里 Equation (3) 定义的是 signed TCD：

```latex
TCD_i = max(...) - max(...)
```

但 Figure 1 和一些文字里又说 “TCD measures the absolute difference”。这会让 reviewer 觉得定义不严谨。

建议统一成：

> We define signed TCD as the per-token contribution difference. Unless otherwise stated, our analyses use absolute TCD, (|\mathrm{TCD}_i|), to measure sensitivity magnitude.

Figure 1 里也可以改成：

> Absolute TCD measures sensitivity magnitude: (|\mathrm{MaxSim}(A)-\mathrm{MaxSim}(B)|)

这样 signed analysis 和 main TCD analysis 就不冲突了。
