# 3. 当前最需要修的核心问题

## 3.1 Query ablation 的 causal claim 过强

这是当前版本最大的科学表述问题。

你现在在 4.1 写：

> To test whether the presence of function words in the query causally drives the overall score sensitivity...

并且结论写：

> confirms that function words are not merely passive post-hoc carriers; their active inclusion ... grammatically amplifies...

但是这个实验的实际 effect 很小：

* SS 从 0.0361 降到 0.0344；
* Ablation Sensitivity Ratio = 1.051×；
* p = 0.013；
* Cohen’s d = 0.0875。

这个可以说是 **small but statistically reliable intervention-style evidence**，但不能说 “confirms” 或 “causally drives”。原因有两个：

第一，ablation 同时改变了 query length、syntax、token composition、normalization context，不是只删除 function-word contribution。第二，d = 0.0875 极小，作为 causal mechanism 证据很弱。

建议改成：

> **Query ablation probe.** To test whether function words merely appear as post-hoc attribution carriers or also affect aggregate score sensitivity, we compare the original function-heavy query with a content-only ablated query. The original query yields slightly higher SS than the ablated query (0.0361 vs. 0.0344; ratio = 1.051×; paired Wilcoxon p = 0.013; d = 0.088). This small but statistically reliable reduction suggests that function-word context contributes to overall sensitivity, although the effect size is modest and the ablation changes query length and syntax as well as function-word content.

这个版本更稳。

另外，这个小实验不一定适合放在 4.1 “Control Validation”。它更像 mechanism/intervention probe。建议移动到 4.6 “Mechanistic Diagnostics” 或 4.2 末尾。如果不移动，也至少把标题改成 **Query ablation probe**，不要叫 intervention 太重。

---

## 3.2 Section 3.3 里 “drives the difference” 仍有一点因果味

3.3 写：

> Aggregate score gaps tell us that a model treats two documents differently, but not which part of the query drives the difference. TCD answers this...

这个 “drives” 仍然暗示 causal attribution。你前后都在强调 TCD 是 exact accounting，不是 causal attribution，所以这里最好统一。

建议改成：

> Aggregate score gaps tell us that a model treats two documents differently, but not where the difference appears in the decomposed query-token score. TCD answers this by accounting for the score gap at the level of individual query-token contributions.

这个小改会让全文更一致。

## 3.3 Abstract 最后一句 “trigger” 稍微强

现在 abstract 说：

> synthetic candidate-pool ranking simulations demonstrate that these score shifts trigger rank displacements and top-k membership changes.

“trigger” 有点决定性。建议改成：

> can produce rank displacements and top-k membership changes

或者：

> are large enough to produce rank displacements and top-k membership changes in synthetic candidate pools.

这样更符合 simulation 的性质。

# 5. 还有几个 reviewer 可能会抓的小点

## 5.1 “all prior retrieval fairness studies” 有点绝对

Related Work 里写：

> Our work differs from all prior retrieval fairness studies by providing token-level attribution rather than aggregate measurement.

这个 “all prior” 容易被 reviewer 挑战。更稳的写法：

> Our work differs from most prior retrieval fairness studies by focusing on exact token-level score decomposition rather than aggregate measurement.

或者：

> To our knowledge, our work is the first to exploit ColBERT’s MaxSim decomposition for token-level score-sensitivity auditing.

后者更精准。

## 5.2 Appendix G 的 “statistically rigorous null result” 仍然略强

Appendix G 里 multiple comparison 后 race-level directional comparisons drops to zero，然后说：

> This statistically rigorous null result reinforces...

这里可以保留，但我建议把 “statistically rigorous null result” 改成：

> This corrected directional analysis reinforces...

因为 “null result” 很容易被理解成你证明了不存在 race effect，但事实上只是当前 setup 下没有 robust evidence。

## 5.3 Computational Details 需要更新总运行时间

你现在新增了 query ablation、200-trial ranking simulation、bootstrap CI、margin bins，但 computational details 仍然只写：

> main ColBERT 22 minutes; each multi-model 5 minutes; naturalistic 7 minutes; total 45 minutes.

现在这个总时间很可能不包括新增 ranking simulation / ablation / bootstrap。建议补一句：

> The synthetic ranking simulations and query-ablation probes add approximately X minutes.

如果没算，可以写：

> Runtime excludes optional plotting and bootstrap resampling.

不要让 reviewer 怀疑实验时间不一致。

# 7. 我会给的正式 review 倾向

**Recommendation: Weak Accept for Findings / Borderline for Main.**

**Strengths:**
The paper proposes a well-motivated and technically clean diagnostic exploiting ColBERT’s decomposable MaxSim scoring. TCD is simple, exact for ColBERT, and clearly distinguished from causal attribution or social harm. The central function-word concentration result is interesting, supported by a large controlled audit, cluster-aware inference, query-format robustness, token-grouping robustness, naturalistic and MS MARCO validations, and ranking simulations. The fairness framing is careful, especially in separating absolute score sensitivity from directional group harm.

**Weaknesses:**
The core experiments remain mostly synthetic/template-based, and the ranking evaluation is synthetic rather than based on an organic candidate retrieval corpus. The new query ablation provides useful intervention-style evidence but has a very small effect size and should not be described as confirming causal mediation. Mechanistic evidence remains suggestive rather than conclusive. Some appendix floats and figure/table ordering need polish.
